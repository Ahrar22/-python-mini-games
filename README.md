# Python Mini Games Collection

A fun and interactive collection of simple, text-based command-line (CLI) games built entirely in Python. This repository serves as a great starting point for beginners to explore basic game logic, loops, and conditional statements in Python.

---

## 🎮 Included Games

All games are located inside the `Game/` directory[span_0](start_span)[span_0](end_span):

* **`Guess.py`:** A classic number guessing game where you try to find the secret number within a limited number of attempts[span_1](start_span)[span_1](end_span).
* **`MathQuize.py`:** A fast-paced math quiz to challenge and test your mental calculation skills[span_2](start_span)[span_2](end_span).
* **`paper.py`:** A digital version of the traditional Rock-Paper-Scissors (Paper) game against the computer[span_3](start_span)[span_3](end_span).
